# Bee A Collector

A fast-paced honey collecting game made in Unity for my first ever game jam!

🎮 **[Play it on itch.io](https://esperauncey.itch.io/bee-a-collector)**

---

## About the Game

Play as a busy bee collecting nectar from flowers and converting it into honey at the hive. Race against the clock to meet your honey goal each round and unlock upgrades to become the ultimate honey producer!

The game starts easy but as rounds progress the honey goal increases and the pressure builds. Efficient pathing and smart timing of your runs to the hive start to matter more and more as you push further.

Every round starts from the exact same position in the exact same level — an intentional design choice that rewards memorisation and invites speedrunning.

---

## Features

- Fast paced nectar collecting gameplay
- Factory conversion system — turn nectar into honey
- Random upgrade system with revealed and mystery options
- Round based progression with increasing honey goals
- Fixed level layout designed for memorisation and speedrunning

## What I Learned

This was my first game jam and I challenged myself to hand craft as much of the art and code as possible on my own. I used Claude.ai purely as a debugging aid. Through this project I learned:

- Unity 2D physics and Rigidbody2D movement
- C# interfaces and how to apply them in Unity
- ScriptableObjects for data driven design
- UI management across multiple canvases
- Scene management and build settings
- Tilemaps for level creation
- Game loop and round management systems
